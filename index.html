About

SangerQuant is a standalone sanger trace viewer and analyzer designed for the quantification of nucleotide mismatches in a batch of samples

This application can handle >100 sanger files in parallel and quantify nucleotide mismatch between samples at any given position

In addition, this program supports common features of sanger trace viewers such as sequence visualization, alignments ...

Key Features

  • Supports .ab1 file format
  • Quantify Nucleotide Mismatches across Samples for Comparison
  • Compare Sequence Quality between Samples to Ensure Data Reliability
  • Export Selected Sequences Into SVG For Quick Embedding Into Scientific Articles
