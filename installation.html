About Installation Tutorial

Installation

Windows
  • Download the Windows installer
  • Double click on .exe file to start the installer
    • Linux / Ubuntu
  • Download Linux AppImage
  • Make the AppImage executable using: chmod +x SangerQuant-linux-x86_64.AppImage
  • Double click on the AppImage or type ./SangerQuant-linux-x86_64.AppImage in terminal
