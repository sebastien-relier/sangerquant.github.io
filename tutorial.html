SangerQuant

Fast Quantification of Sequence Mismatch from Sanger files

About Installation Tutorial

Tutorial

Import data

  • Click on "File" of the menubar and click on "Import"
  • Select one or more .ab1 files from sanger traces and click open. [Press shift + left click for multiple selections]
  • Press *open* to import the files

Explore Traces

  • Select a single file from the list on the left to display the corresponding trace
  • Click on the scrollbar above the trace to navigate along the sequence | Alternatively, use the mouse wheel
  • To navigate directly to a specific sequence of interest, type the query sequence in the text box below the trace

Mismatch quantification

Peak height quantification from single traces
  • Left-click on a peak from the trace to quantify peak heights, raw value will appear below the sample list on the bottom left
Batch quantification from multiple files
  • Navigate to Analyses > Quantify mismatch in the menu
  • In the popup window, enter your sequence in the top textbox. | Replace the nucleotides you want to quantify with an "N". ie. GCATGGCCGTTCTT > GCATGGCNGTTCTT
  • From the list, use your left mouse button to select the samples you want to analyze
  • Select the mutation to quantify from the combobox
  • Press Preview to display the quantification results in the table
  • To export: Use the Export button to save all data | right-click on specific rows, select Copy, and paste into your favorite worksheet software

De novo mismatch identification

  • Navigate to Analyses > Find mismatches in the menu
  • From the list, use your left mouse button to select the samples you want to analyze
  • Paste your reference sequence in FASTA format into the textbox below the sample list
  • Use your scroll wheel to zoom in or out
  • Left click on a bar of interest to display its position in the main window
  • Click the Export button to save the plot

Edit the Sanger Traces

Trim low quality sequence
  • Navigate to Edit > Trim
  • From the list, use your left mouse button to select the traces you want to trim
  • A plot showing Peak Quality (Phred Score) vs. Peak Position will appear. Higher values indicate better quality
  • Left-click on the red vertical lines and drag them along the x-axis | The area between the red lines will be kept, while the outer shaded areas will be trimmed.
  • Click Trim to apply the changes
  • (Optional) To undo, click Untrim.
